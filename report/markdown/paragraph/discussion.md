Fossil SCM
---
剛開始接觸都不太熟，甚至連fossil也不知道是甚麼。但經過幾個禮拜老師的教學後，不但都能了解，也能實際應用。
Fossil SCM相關指令:
fossil init 倉儲名稱.fossil
fossil ui
fossil open
fossil clone
fossil add .
fossil commit -m --no-warnings
fossil update
